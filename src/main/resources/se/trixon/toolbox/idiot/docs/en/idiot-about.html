IDIOT (IDIOT Downloads Images On Time) downloads images from web cams with regularity.

Toolbar

The toolbar has a number of buttons that control program execution.
  1. Schedule downloads - turn on and off schedule.
  2. Download now - download image immediately.
  3. Open Directory - opens the directory where the images are stored.
  4. Add - adds a new job.
  5. Clone - clone the selected job.
  6. Edit - edit the selected job.
  7. Remove - remove the selected job.
  8. Remove all - remove all jobs.
  9. Help - displays this help.

Job

Name specifies the name of the job.

Description is the descriptive text that appears in the job list.

URL is the path to the picture on the internet.

Destination, path with the directory and base file name. Downloaded images gets a suffix with date and time.

Active, indicates whether the scheduler should run this job or not.
